## Most Relevant Papers for Our Research

The following papers are most directly relevant to the hypothesis that LLMs can expose under-researched topics:

1. **GAPMAP** (2510.25055) - Demonstrates LLM capability in identifying explicit and implicit knowledge gaps
2. **Knowledge Gaps/Silos** (2406.03921) - Provides framework for identifying isolated research areas
3. **Big Tech Funding** (2512.05714) - Shows how funding drives research attention, creating biases
4. **FIND NSF** (2510.10336) - Provides data infrastructure for funding-publication analysis
5. **Can LLMs Generate Novel Ideas** (2409.04109) - Empirical evidence that LLMs can identify novel directions
